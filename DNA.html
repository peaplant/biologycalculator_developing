Tm value of DNA

  DNA sequence (only A,T,G,C are allowed)

Length of DNA


Formula : 

For sequence less than 14 nucleotides : Tm= (nA+nT)*2+(nG+nC)*4
For sequences longer than 13 nucleotides : Tm= 64.9 + 41*(nG+nC-16.4)/(nA+nT+nG+nC)

nA : number of A, nT : number of T, nG : number of G, nC : number of C
*Tm calculation varies between sites. So you have to optimize it yourself.
